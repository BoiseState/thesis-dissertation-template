## Description

---

Customized style class file and template for Computer Science Masters thesis/project or PhD dissertation reports and proposals. Approved by the Graduate College at Boise State University.

## How to use

---

1. Select the correct template for your task.
  (A) For writing a thesis, dissertation, or project *report*, use`bsu-thesis-dissertation-template.tex`.
  (B) For writing a thesis, dissertation, or project *proposal*, use `bsu-proposal.tex`.
2. Leave `bsu-cs.cls` (the LaTeX style file) in the same folder as your `.tex` document file.
3. You may want to create a copy of the template to preserve it while working on your document. One LaTeX convention is to name your main working document `main.tex`.

The `bsu-thesis-dissertation-template.pdf` and `bsu-template.pdf` files show finished/rendered PDF version of their respective template LaTeX (`.tex`) files.

## LaTeX editors

---

- You can upload these files to [Overleaf](https://www.overleaf.com/) to use web-based LaTeX.
- On Linux, you can use [kile](https://kile.sourceforge.io/) for a nice GUI for LaTeX. For the command line, use `pdflatex` to generate PDF properly from LaTeX document.
- On Windows, you can get the [MikTEX](miktex.org) package and use the style files provided here.
- On Mac, [TeXShop](https://pages.uoregon.edu/koch/texshop/obtaining.html) is one option.
